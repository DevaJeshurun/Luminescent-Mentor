# 🚀 CodeMentor AI — AI Programming Tutor & Mock Interview Platform

CodeMentor AI is a **full-stack AI-powered learning platform for programming students**.

It helps students:

* understand code in any programming language
* get line-by-line AI explanations
* practice DSA questions
* take mock interviews
* solve Java MCQs
* improve weak topics
* prepare for placements

The platform is designed as a **student-first coding tutor SaaS**, combining:

* AI chatbot tutoring
* DSA practice
* Java learning
* coding playground
* mock interview rounds
* analytics

# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/DevaJeshurun/Luminescent-Mentor.git
cd code-mentor-ai
```

## 2️⃣ Backend Setup

```bash
cd backend
mvn spring-boot:run
```

Backend runs on:

```text
http://localhost:8081
```

## 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
